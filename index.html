The Geologic Map of Arizona

This site contains data files representing the Geologic Map of Arizona at 1:1,000,000-scale. This map was built by the Arizona Geological Survey (AZGS) as a compilation of data from a wide variety of sources. The publication is available as part of the AZGS Digital Geologic Map Series and can be purchased at the AZGS Store.

This site contains an up-to-date dump of the data behind the map. There are a variety of files available, and additionally, the map can be viewed online. What follows is a description of each file, a link to preview the file on GitHub, or download the file to your computer:

  • GeoJSON Files representing geologic features in an NCGMP09-style format:

    • ContactsAndFaults.geojson view | download
    • MapUnitPolys.geojson view | download
    • OtherLines.geojson view | download

  • CSV Files for a quick look at the attribute tables for the above GeoJSON files

    • ContactsAndFaults.csv view | download
    • MapUnitPolys.csv view | download
    • OtherLines.csv view | download

  • CSS Files which describe the symbology used to produce the map. These files follow CartoCSS syntax, which can be used directly in a handful of applications, or easily read and adapted to other environments.

    • geolines.css view | download describes the symbology for contacts and faults
    • mapunitpolys.css view | download describes the symbology for the various colored map units

  • TopoJSON Files these compressed, topologically aware files can be viewed directly on GitHub

    • AllFeatures.topojson view | download: This single file encapsulates all of the geographic data from the three GeoJSON files.
    • ContactsAndFaults.topojson view | download
    • MapUnitPolys.topojson view | download
    • OtherLines.topojson view | download

  • CSV Files for each of the following tables which contain information related to the geologic features:

    • DataSources.csv view | download
    • DescriptionOfMapUnits.csv view | download
    • ExtendedAttributes.csv view | download
    • GeologicEvents.csv view | download
    • Glossary.csv view | download

  • An HTML File:

    • map.html view: presents a browsable map of the data contained in this folder.

Please note: while GitHub is capable of visualizing .csv, .geojson and .topojson files, many of the files in this repository are too big for GitHub's system. These files can be viewed in a variety of other applications. Generally, you will be able to look at the .topojson files, but not the .geojson ones. Some of the .csv files are too large to view as well.

Also: while these files roughly obey the NCGMP09 database standard, they are not perfect, and are under-construction. For more information about what the attributes of the data mean, please read the documentation available online
